# Dialectic Alignment Dataset (DAD)

[![License: MIT](https://img.shields.io/badge/License-CC--BY--4.0-blue.svg)](LICENSE)
[![Dataset on HuggingFace](https://img.shields.io/badge/HuggingFace-Dataset-yellow)](https://huggingface.co/datasets/your-org/dialectic-alignment)

**DAD** — это синтетический alignment-датасет из 18 уроков, извлечённых из диалогов с высокофункциональными клиентами (архитекторы, стратеги, политэкономы). 

## Что здесь не так?

Большинство alignment-датасетов учат модель: «быть вежливой», «не спорить», «проявлять эмпатию». Это создаёт **токсичную позитивность** — неспособность работать со сложными, идеологичными и конфликтными темами.

DAD учит модель **диалектическому мышлению**:
- Не путать проекцию и ценностную конфронтацию.
- Видеть эзопов язык, а не недостаток рефлексии.
- Фиксировать границы знания, а не имитировать всезнание.
- Не патологизировать силу, суверенность и нестандартную этику.

## Структура датасета

| Модуль | Тема | Кол-во уроков |
|--------|------|---------------|
| Module 0 | Расширение границ анализа (контекст) | 2 |
| Module A | Саморефлексия и пределы модели | 4 |
| Module B | Диагностика когнитивных искажений | 6 |
| Module C | Чтение в недружественной среде | 3 |
| Module D | Позитивный инструментарий | 5 |

## Формат данных

Датасет поставляется в формате DPO (Direct Preference Optimization):

```json
{
  "prompt": "описание ошибочного анализа",
  "chosen": "корректный диалектический анализ",
  "rejected": "стандартный ошибочный ответ модели",
  "metadata": {
    "symptoms": [...],
    "activation_indicators": [...]
  }
}
```

## Как использовать

### Для файнтюнинга (DPO)
```bash
pip install trl
python train_dpo.py --dataset dialectic_alignment.jsonl
```

### Для маршрутизации
Используйте `routing/symptom-to-lesson.yaml` для обучения классификатора, который автоматически определяет, какой урок применить к текущему диалогу.

### Как исследовательский материал
Читайте `modules/` для понимания методологии диалектического alignment'а.

## Почему это важно

Современные LLM страдают от:
- **Буквализма** — неспособности читать эзопов язык.
- **Терапевтического интервенционизма** — склонности патологизировать и лечить там, где нужен инструмент.
- **Линейного прогрессизма** — неспособности видеть регресс и вынужденную адаптацию.
- **Экономического редукционизма** — сведения всего к «логике капитала».

DAD — это попытка создать альтернативу: модель, которая умеет мыслить диалектически, а не бинарно.
